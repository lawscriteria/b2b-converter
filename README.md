# b2b converter
## b2b converter converts any given integer from base n to base k where (n,k) Ð„ {2,3,4,5,6,7,8,9,10}. Both n & k must be specified with an <b>indicator</b>, which is a single-letter representation of your number's numerical system or 'base'.

## Indicators
<ul>
  <li><b>binary: 'b'</b></li>
  <li><b>ternary: 't'</b></li>
  <li><b>quaternary: 'q'</b></li>
  <li><b>quinary: 'p'</b></li>
  <li><b>hexary (seximal): 'h'</b></li>
  <li><b>septimal: 's'</b></li>
  <li><b>octal: 'o'</b></li>
  <li><b>nonary: 'n'</b></li>
  <li><b>decimal: 'd'</b></li>
</ul>

<hr>

# e.g:
## f(-34100000p, d) => -300000
## f(314n, s) => 514
## f(37d, b) => 100101
